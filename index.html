What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper, or in otherr media.
  3. Css is not necessary CSS can saves a lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheets are stord in CSS files.

CSS can be added to HTML documetns in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the style in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to deomstrate, and easier for you to try it yourself.


When Comparing CSS class bs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and its used by JavaScript


In CSS, selectros are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.


There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target element to which a style should be applied.
